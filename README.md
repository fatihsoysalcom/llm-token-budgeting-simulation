# LLM Token Budgeting Simulation

This example demonstrates the concept of token budgeting for Large Language Models (LLMs). It uses the `tiktoken` library to accurately count tokens in different text inputs, simulates a hypothetical token budget, and shows how to check if a prompt fits within the budget. It also illustrates a basic truncation strategy for prompts that exceed the allocated token limit, highlighting the importance of managing token usage for cost and performance optimization in AI engineering.

## Language

`python`

## How to Run

1. Install the required library: `pip install tiktoken`
2. Run the script: `python main.py`
